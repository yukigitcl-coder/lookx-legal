サービスについて

LOOKX はどんなサービスですか？
写真をアップロードすると、AI が「ヘアスタイル / スキンケア / メイク・グルーミング / 全体的な印象」の4つの観点からあなたの伸びしろを評価し、改善のための具体的なアドバイスを返すアプリです。
医療行為や診断ですか？
いいえ、医療行為ではありません。本サービスは美容・スタイルの一般的なアドバイスを提供するもので、医学的診断や治療を行うものではありません。気になる症状がある場合は皮膚科などの専門医にご相談ください。
人物の顔以外の写真でも診断できますか？
本サービスは人物の顔や全体の見た目を対象としています。風景・動物・キャラクターなど人物が写っていない写真の場合、診断はできませんが、無料お試しの場合は無料枠は消費されません。別の写真でお試しください。

料金・課金について

料金プランを教えてください
無料お試し（広告視聴後・初回1回限り・1領域のみ）、単発診断 ¥400、週額プラン ¥350/週、月額プラン ¥1,200/月（初月50%OFF、2ヶ月目以降 ¥2,400/月）、年額プラン ¥5,800/年 をご用意しています。詳細はアプリの「プラン選択」画面をご確認ください。
サブスクリプションを解約したいです
解約はいつでも可能です。
・iOS版: 設定アプリ → Apple ID → サブスクリプション
・Android版: Google Play ストア → アカウント → お支払いと定期購入 → 定期購入
・Web版: アプリ内の設定画面から
解約手続き後、現在の課金期間が終了するタイミングでサービスが停止します。期間中は引き続きご利用いただけます。
途中解約した場合、日割りで返金されますか？
日割り返金はありません。解約後も現在の課金期間が終了するまではプレミアム機能をご利用いただけます。例えば月額プランを月初に解約した場合でも、その月末まで利用可能です。
無料お試しを何度も使いたいです
無料お試しは1ユーザーにつき初回1回限りとなっています。複数アカウント作成による不正利用は規約違反となりますのでご遠慮ください。
広告を見ても無料診断できません
広告再生は完了しましたか？ 途中でスキップした場合は無料診断が付与されない場合があります。最後まで視聴いただきましたらアプリを再起動してお試しください。それでも解決しない場合は lookx0520@gmail.com までご連絡ください。

プライバシー・データについて

アップロードした写真はどこに保存されますか？
デフォルトでは保存されません。解析処理のため一時的に外部の AI API（Anthropic Claude / Google Gemini）に送信され、処理後即時破棄されます。あなたが明示的に「結果を保存する」を選んだ場合のみ、暗号化通信経由で当社のストレージに保存されます。
写真は AI の学習に使われますか？
いいえ、使われません。Anthropic / Google の API 規約上、API 経由で送信された画像はモデルの学習には使用されません。
アカウントを削除したいです
アプリ内の設定画面から「アカウント削除」を選ぶか、lookx0520@gmail.com までご連絡ください。削除後、保存済みの写真・診断結果・アカウント情報はすべて消去されます。なお、課金履歴は法令で定められた期間（最大7年）保管されます。
他人の顔写真をアップロードしても良いですか？
本人の事前同意がない限りアップロードしないでください。プライバシー権・肖像権の侵害に該当する可能性があり、規約違反となります。アップロードの結果生じた一切の責任はアップロードしたユーザーが負うものとします。

未成年の利用について

何歳から利用できますか？
13歳以上を対象としています。18歳未満の方は保護者の同意を得てご利用ください。

不具合・トラブル

解析中にエラーが出ました
ネットワーク状況をご確認のうえ、しばらく時間をおいて再試行してください。それでも繰り返し失敗する場合は、本サービスの現状については ステータスページ でご確認いただけます。
スコアが低くて落ち込みます
本サービスのスコアは「現在の整え方の伸びしろ」を表すもので、容姿の優劣を示すものではありません。低い数値は「改善できる余地が大きい」という意味であり、あなたの価値を否定するものではありません。提示される「具体的なアクション」を1つずつ試してみてください。
